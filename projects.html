Minigolf

- Description:

MiniGolf is a classic 2d game implemented in Python with the use of pygame library and pymunk physics engine. Pygame is responsible for main game loop, rendering graphics and listening for events. Pymunk provides physics simulation of the game which goes in the "background". You can play both single and local muliplayer for at most five players.

Compiler

- Description:

This project is an implementation of Compiler for small imperative language called Imperator. It is written in C++ language with the use of FLEX and Bison. FLEX generates scanner, which groups input characters into lexical units / tokens. Bison generates parser, which groups tokens into syntactical units and for each unit recognized there is an action implemented in C++, which uses back of the compiler to generate assembler code.
